# LinguaLAN - Electron Version (Legacy)

This is the original Electron-based version of LinguaLAN.
It has been superseded by the [Tauri v2 version](../) which is ~10x smaller (~5.6 MB vs ~75 MB).

## How to run

```bash
cd electron-version
npm install
npm start
```

## Why Tauri?

The migration from Electron to Tauri v2 reduced the binary from ~75 MB to ~5.6 MB while keeping all features. See the [main README](../README.md).
